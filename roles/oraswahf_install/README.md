oraswahf_install
================

This role installs Autonomous Health Framework from Oracle.

Requirements
------------

The ZIP Archive with AHF from Oracle needs to downloaded before.

Role Variables
--------------

ahf_filename

Filename from ZIP archive from Oracle.
